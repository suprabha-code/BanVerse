# Banverse Project

##  Overview

Banverse is an **AI-powered student engagement and management platform** designed to enhance campus interaction, automate grievance handling, and provide personalized recommendations using intelligent algorithms.

---

## Features

* AI-based recommendation system
* Student dashboard & analytics
* Club and event management
* Authentication system (Login/Signup)
* Chatbot integration
* Notifications & announcements
* Post creation & interaction

---

## Tech Stack

### Frontend

* React.js
* HTML, CSS, JavaScript

### Backend

* Node.js
* Express.js

### AI Backend

* Python
* Machine Learning models

### Database

* MongoDB

## ⚙️ Installation & Setup

### 🔹 1. Clone Repository


git clone https://github.com/your-username/banverse-project.git
cd banverse-project


---

### 🔹 2. Install Frontend Dependencies


npm install


---

### 3. Run Frontend


npm start


---

### 🔹 4. Setup Backend


cd banverse-backend
npm install
node server.js


### 🔹 5. Setup AI Backend


cd aibackend
pip install -r requirements.txt
python app.py

---

## 🔐 Environment Variables

Create a `.env` file in backend folders and add:

MONGO_URI=your_mongodb_connection
PORT=5000
